# 版本记录

每次界面或功能调整后，都会保存为一个完整版本。版本目录在 `versions/`，可用版本号或完整目录名恢复。

## 使用

```bash
node version-manager.mjs snapshot "更动内容名称"
node version-manager.mjs list
node version-manager.mjs restore v0001
```
